# mcp-agent-skills

[Agent skills](https://agentskills.io/home) for building, testing, and learning about [MCP](https://modelcontextprotocol.io/).

| Skill | Description |
|-------|-------------|
| [mcp-server-ts](./skills/mcp-server-ts) | Build MCP servers in TypeScript with tools, resources, and prompts |
| [mcp-client-ts](./skills/mcp-client-ts) | Build MCP clients in TypeScript to connect to MCP servers |
| [mcp-apps-ts](./skills/mcp-apps-ts) | Build interactive HTML UIs for MCP tools using the MCP Apps extension (SEP-1865) |
